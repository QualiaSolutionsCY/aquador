## Root Cause (1/2)
The `products` table stores each size as a separate row sharing a base slug (see `getProductVariantGroup`). All listing routes fed raw rows to `ProductGrid`, so each size rendered as its own card (e.g. "Imagination by Louis Vuitton" + its -100ml row). Shipping showed "free shipping" with no threshold because `FREE_SHIPPING_THRESHOLD`/`DELIVERY_FEE` were both `0`.
