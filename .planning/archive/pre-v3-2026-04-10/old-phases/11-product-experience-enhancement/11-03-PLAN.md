Create new ProductCard component for consistent luxury product presentation across site:

**Component interface:**
```typescript
interface ProductCardProps {
  product: LegacyProduct | Product; // Union type to handle both formats
  priority?: boolean;
  variant?: 'default' | 'compact';
}
```

Where:
- `LegacyProduct` has: { inStock: boolean, salePrice?: number, ... }
- `Product` (Supabase) has: { in_stock: boolean, sale_price?: number, ... }

**Normalize props inside component:**
```typescript
// Normalize property names to handle both formats
const inStock = 'in_stock' in product ? product.in_stock : product.inStock;
const salePrice = 'sale_price' in product ? product.sale_price : product.salePrice;
```

**Structure:**
- Use Link wrapper to product detail page (/products/[slug])
- ProductImage with variant="card" (400x500, 4:5 aspect ratio)
- Sale badge: positioned absolute top-3 right-3, rounded-full, bg-red-500/90 backdrop-blur-sm
- Brand label: text-[clamp(0.625rem,0.5625rem+0.3125vw,0.75rem)] tracking-[0.15em] uppercase text-gold-500
- Product name: text-[clamp(1rem,0.875rem+0.625vw,1.25rem)] font-playfair font-medium tracking-tight text-white
- Price: text-[clamp(1.125rem,1rem+0.625vw,1.5rem)] font-playfair font-medium text-gold-600
- Sale price strikethrough: text-gray-500 line-through text-sm

**Styling:**
- Card background: bg-dark-lighter/80 backdrop-blur-sm
- Border: border border-gold-500/10 hover:border-gold-500/30
- Border radius: rounded-2xl
- Padding: p-4 md:p-5
- Shadow: shadow-lg shadow-black/20 hover:shadow-xl hover:shadow-gold-500/10
- Transition: transition-all duration-300 ease-out
- Hover state: transform hover:-translate-y-1 (subtle lift)

**Out of stock:**
- Image overlay: absolute inset-0 bg-black/40 with "Coming Soon" badge
- Opacity: overall card opacity-75

**Variant: compact** (for related products):
- Smaller padding: p-3 md:p-4
- Smaller text sizes (reduce clamp() ranges by ~15%)
- Smaller ProductImage if needed

**Accessibility:**
- Link has proper aria-label: "View {product.name}"
- Focus state: focus-visible:ring-2 focus-visible:ring-gold-500 focus-visible:ring-offset-2
- All interactive states keyboard accessible
