**State management:**


**Desktop click-to-zoom:**
- Add handleImageClick:

  - Transition: transform 0.3s ease-out
  - Cursor: cursor-zoom-out
  - Backdrop: fixed inset-0 bg-black/40 backdrop-blur-sm
- Add handleMouseMove to update zoomPosition during zoom (for pan effect)
- ESC key listener: if (e.key === 'Escape' && isZoomed) setIsZoomed(false)

- Add touch event handlers:

- Add useEffect to reset zoom when selectedIndex changes (new image selected)

**Navigation controls:**
- Disable navigation arrows when isZoomed (add disabled={isZoomed} prop)
- Disable swipe handlers when isZoomed (add condition in swipe handler)
- Maintain Framer Motion slide transitions (keep existing variants)
