  <name>Task 1: Create cart validation module with Zod schema and price verification</name>
  <files>src/lib/validation/cart.ts</files>
  <action>
Create a new validation module at `src/lib/validation/cart.ts` containing:

1. **Zod schema for CartItem validation** (`cartItemSchema`):
   - `productId`: non-empty string
   - `variantId`: non-empty string matching pattern `{productId}-{productType}-{size}`
   - `quantity`: positive integer, min 1, max 100
   - `name`: non-empty string, max 200 chars
   - `image`: string (URL format preferred but not required - some products may have local paths)
   - `price`: positive number
   - `size`: enum matching ProductSize ('10ml' | '50ml' | '100ml' | '150ml')
   - `productType`: enum matching ProductType ('perfume' | 'essence-oil' | 'body-lotion')

2. **validateCartPrices function** signature:
```typescript
export function validateCartPrices(items: CartItem[]): {
  valid: boolean;
  errors?: Array<{ productId: string; reason: string }>;
}
```

Implementation:
- For each item, call `getProductById(item.productId)` from `@/lib/product-service`
- Verify product exists (if not: error "Product not found")
- Compare `item.price` against product's `salePrice || price`
- Allow 1-cent tolerance for floating-point rounding (Math.abs(itemPrice - catalogPrice) <= 0.01)
- Verify `item.productType` matches product's `productType`
- Verify `item.size` matches product's `size`
- Return `{ valid: false, errors: [...] }` if any item fails
- Return `{ valid: true }` if all items pass

Import from: `@/lib/product-service` (getProductById), `@/types/cart` (CartItem), `@/types/product` (ProductType, ProductSize)
Use: Zod for schema validation

```typescript
import { z } from 'zod';

```

2. **Add Zod validation** after parsing body (before line items creation):
```typescript
// Validate cart items schema
const validationResult = z.array(cartItemSchema).safeParse(items);
if (!validationResult.success) {
  return NextResponse.json(
    { error: 'Invalid cart data', details: validationResult.error.errors },
    { status: 400 }
  );
}

// Validate prices against catalog
const priceValidation = validateCartPrices(items);
if (!priceValidation.valid) {
  return NextResponse.json(
    { error: 'Price mismatch detected', details: priceValidation.errors },
    { status: 400 }
  );
}
```

```typescript
// SECURITY: Validate cart items and verify prices against server-side catalog
```
